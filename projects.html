Quad Tree

Using a simple 2D framework, I implemented quadtree-based spatial partitioning to optimize collision detection for 1000 circles. Instead of checking every circle against every other one, which gets expensive fast, I used a quadtree to divide the space into regions and quickly narrow down which circles could potentially collide. This cut down the number of collision checks per frame significantly while keeping the detection accurate.

Dungeon Generation

Using a simple 2D framework, I implemented dungeon generation code. With configurable numbers to customize the final result. Uses AABB collision to seperate boxes. Delaunay Triagulation and Minimum spanning tree to create hallways between selected rooms.
